# Core Gameplay Pillars

1. **Loadout-Driven Identity**
   - Equipped gear determines card pool and tactical identity.

2. **Readable Tactical Combat**
   - Grid, turn order, movement, line-of-sight, and effect timing are transparent.

3. **Procedural but Legible Spaces**
   - Dungeons generated with dungeoneer should remain tactically understandable.

4. **Build Iteration Loop**
   - Each encounter creates new decisions: keep, swap, or pivot equipment/cards.

5. **Deterministic Rules Engine**
   - Same seed + same inputs => same outcomes.


6. **Rank-Up Identity Layer**
   - Rank thresholds unlock major evolution choices (mutations/branches), not just numeric stat increases.
