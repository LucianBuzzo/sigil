# MVP Scope

## Must-Have

- Single tactical encounter loop
- Dungeoneer-generated map adapted to grid
- 3 playable classes
- 8-12 items with card grants
- 20-30 cards
- Deterministic turn resolution
- One vertical rank-up flow (Rank I -> Rank II)
- At least 2 rank mutation choices with visible gameplay impact

## Nice-to-Have (Post-MVP)

- Multi-encounter run persistence
- Meta progression
- Expanded status/effect taxonomy
- Enemy AI sophistication tiers
- Full multi-rank progression trees
- Extensive rank-branch balancing

## Explicitly Out-of-Scope (MVP)

- Online multiplayer
- Real-time combat
- Open-world traversal
