# Roadmap

## Phase 0 — Scaffold (current)

- Project name and setting established
- Foundation docs created

## Phase 1 — Core Simulation Vertical Slice

- Grid + turn order + card resolution basics
- One generated map playable in headless simulation

## Phase 2 — Client Prototype

- Basic board rendering
- Hand/deck/discard UI
- Targeting + action execution

## Phase 3 — Build Loop

- Equip/unequip flow
- Item-driven deck regeneration
- Encounter completion + loot replacement

## Phase 3.5 — Rank-Up Vertical Slice

- Add rank data model (rank, trials, mutations)
- Implement Rank I -> Rank II progression trigger
- Add rank-up choice UX with at least 2 mutation branches
- Persist rank choice and apply gameplay deltas

## Phase 4 — Balance + Content Expansion

- Add classes/cards/enemies
- Expand rank trees and mutation variants
- Introduce simulation harness for batch balancing
